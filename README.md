# Proyecto Animales 🐾

Este proyecto simula una jerarquía de clases en Java usando:
- Herencia
- Interfaces
- Polimorfismo
- Composición

## Objetivo
Practicar el uso de Git con ramas y resolución de conflictos en un proyecto Java orientado a objetos.

## Estructura
- Animal (clase abstracta)
- Mamifero
- Leon
- Ballena
- Gacela
- Interfaces: Depredador y Presa
